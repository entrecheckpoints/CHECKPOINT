# Checkpoint Android 1.0.0 — Live Magazine Edition

Checkpoint 1.0 transforma la pantalla Inicio en una revista gamer dinámica construida con los precios reales del usuario.

## Destacado

- Portada diaria generada desde juegos, eventos, objetivos e historial.
- Titulares dinámicos y juego destacado mediante Deal Score.
- Número de edición y barcode automáticos.
- Best Deals Today.
- Price Pulse.
- Inicio más visual, mientras las herramientas avanzadas permanecen organizadas en sus secciones.
- Temas Editorial, After Dark, Frutiger Aero y Arcade Neon compatibles con la nueva portada.
- Workflow para publicar el APK directamente en GitHub Releases.

## Compatibilidad

La actualización conserva la base de datos y los respaldos de 0.3.1. Requiere la misma firma para instalarse encima de una versión anterior.
